# Pet-проект "Amazon"
## Задача проекта:
Оценить влияние скидок на ключевые метрики маркетплейса: выручку, средний чек, удержание клиентов и уровень возвратов, а также определить, в каких товарных категориях скидки работают наиболее эффективно. На основе анализа сформулировать рекомендации для бизнеса по оптимизации скидочной стратегии.
## Описание датасета:
Этот набор данных содержит 100 000 синтетических транзакций продаж в электронной коммерции в стиле Amazon  в период c 01.01.2020 по 29.12.2024, разработанных для того, чтобы максимально точно имитировать реальное поведение покупателей в онлайн-торговле. Благодаря 20 четко структурированным столбцам, он содержит подробную информацию о клиентах, продавцах, товарах, ценах, платежах, логистике и результатах заказов.
### В набор данных входят:
* OrderID - Идентификатор заказа
* OrderDate - Дата заказа
* CustomerID - Идентификатор клиента
* CustomerName - Имя клиента
* ProductID - Название продукта
* ProductName - Название продукта
* Category - Категория
* Brand - Бренд
* Quantity - Количество
* UnitPrice - Цена за единицу товара
* Discount - Скидка
* Tax - Налог
* ShippingCost - Стоимость доставки
* TotalAmount - Общая сумма
* PaymentMethod - Способ оплаты
* OrderStatus - Статус заказа
* City - Город
* State - Штат
* Country - Страна
* SellerID - Идентификатор продавца
